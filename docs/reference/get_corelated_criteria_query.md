# Get correlated criteria query for a criteria list item (Criteria + StartWindow + Occurrence)

Get correlated criteria query for a criteria list item (Criteria +
StartWindow + Occurrence)

## Usage

``` r
get_corelated_criteria_query(
  correlated_item,
  event_table,
  index_id,
  cdm_schema
)
```

## Arguments

- correlated_item:

  Correlated criteria item.

- event_table:

  Event table expression.

- index_id:

  Index id.

- cdm_schema:

  CDM schema name.
