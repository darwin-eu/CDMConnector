# Get codeset join expression

Get codeset join expression

## Usage

``` r
get_codeset_join_expression(
  standard_codeset_id,
  standard_concept_column,
  source_codeset_id,
  source_concept_column
)
```

## Arguments

- standard_codeset_id:

  Integer or NULL

- standard_concept_column:

  Column name

- source_codeset_id:

  Integer or NULL

- source_concept_column:

  Column name

## Value

SQL JOIN clause
