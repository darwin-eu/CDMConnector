# Collect CDM table names used by cohort objects (for \##\*\_FILTERED)

Collect CDM table names used by cohort objects (for \##\*\_FILTERED)

## Usage

``` r
collect_batch_used_domains_from_cohorts(cohort_list)
```

## Arguments

- cohort_list:

  List of cohort expression lists.

## Value

Character vector of CDM table names (e.g. "DRUG_EXPOSURE").
