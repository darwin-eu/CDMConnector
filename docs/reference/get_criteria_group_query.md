# Get criteria group query

Get criteria group query

## Usage

``` r
get_criteria_group_query(
  group,
  event_table,
  cdm_schema = "@cdm_database_schema"
)
```

## Arguments

- group:

  Criteria group (Type, CriteriaList, Groups, etc.).

- event_table:

  Event table expression.

- cdm_schema:

  CDM schema name.
