# Get value from list with fallback keys (PascalCase/camelCase)

Get value from list with fallback keys (PascalCase/camelCase)

## Usage

``` r
get_key(obj, keys, default = NULL)
```

## Arguments

- obj:

  List

- keys:

  Character vector of possible key names

- default:

  Default value if not found

## Value

Value or default
